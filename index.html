Apa itu Github

GitHub adalah sebuah aplikasi yang berbasis Web dengan Version Control System, yang menyediakan sebuah layanan yang memungkinkan kita untuk menyimpan project kedalam Repository secara gratis. Repository itu sendiri adalah merupakan tempat untuk menyimpan File yang isinya berupa Source Code.

GitHub juga memberikan layanan cloud untuk menyimpan dan mengelola project/repository git. Karena bersifat online, kita meng-edit sebuah repository/project secara bersamaan dengan orang lain di tempat yang berbeda. Oleh karena itu, platform ini sangat membantu tim project dalam menyusun suatu folder yang berisikan files terkait pemrograman.

Banyak programmer bekerja sama dalam satu proyek. GitHub memudahkan mereka untuk berbagi kode dan melihat perubahan yang dilakukan oleh rekan satu tim. GitHub juga berfungsi sebagai portofolio online. Kamu bisa menunjukkan kemampuan programmingmu kepada orang lain, misalnya calon perusahaan tempat kamu ingin bekerja.
